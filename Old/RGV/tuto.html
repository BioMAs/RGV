RGV basic usage
Displaying information about specific gene
  • Step 1 - Enter gene name or synonym in the search bar in RGV home page and validate by clicking on "go" button.
    Valid names include gene symbol and synonyms, transcript IDs (Ensembl, RefSeq) and gene IDs (NCBI, Entrez Gene, Ensembl) and are not case sensitive.
  • Step 2 - In the result page, corresponding entries from all species indexed in RGV are proposed.
    As RGV integrates information from NCBI and Ensembl, genes can be identified in duplicate. Click on the information link to have a direct access to the entry of the corresponding database.
  • Step 3 - Select your entry in the Chromosome + Link field.
Crosslinking RGV
  • RGV has a generic URL that allows you to access the results page directly.
  • Customize the URL with gene name and refine your query with a precise species if necessary.
    Valid names include gene symbol and synonyms, transcript IDs (Ensembl, RefSeq) and gene IDs (NCBI, Entrez Gene, Ensembl). Valid genomes include generic species names (Human, Mouse, Rat, Dog..) as well as precise genome releases (hg19, mm10...).










RGV genome browser (JBrowse)
Display specific gene
  • Step 1 - Enter gene name in the search bar and validate.
    Valid names include gene symbol and synonyms and are not case sensitive.
  • Step 2 - In a result windows, choose your displayed annotation. Click on "Show" or "Go" to display your gene.As RGV integrates information from NCBI and Ensembl, genes can be identified in duplicate.
  • Step 3 - To have an overview of your gene region, use the zoom-in or zoom-out button.
Selecting tracks
  • Step 1 - Use the left menu "Available tracks" to choose your tracks. To show the different tracks, use the collapse menu.
  • Step 2 - Drag and drop selected feature, to reorganized displayed tracks, .
    Note that the “Available Tracks” menu is organized so that the same tracks could have been identified by the “Technologies” tab, by the “Biological topics” tab, by "Species" tab or by "Publications" tab.
  • Step 3 - To delete a track, click on "close" button near of name
Tracks information
  • Step 1 - For genes or variant tracks, you can display an information box by clicking on selected gene. A link to PubMed and ClinVar/GWAS database are available for variant tracks.
  • Step 2 - For expression tracks, the expression value appear when cursor browse the track.
  • Step 3 - To display information box of expression tracks, click on down arrow near of the name and select "About this track". For each tracks, a PubMed link to associated publication is available.
Import your data on genome browser
  • Step 1 - Click on File > Open on the head menu.
  • Step 2 - Files could be opened by drag and drop file in the dedicated area. Files could be also opened by clicking on select files button or by copy / paste a URL.
  • Step 3 - If the file extension is not recognized, pleased defined it in Files and URLs section.
    File accepted are : GTF, GFF3, BigWig, Bam, Bamindex or VCF.
  • Step 4 - Configure the display track. For an advance users, use edit configuration button.

















RGV working environment (Galaxy)
Login
  • Step 1 - In order to have access to files history, finished job alert and more, you have to register to Galaxy. For that select register in user tab.
  • Step 2 - Fill out the form
  • Step 3 - Login in user tab.
Import files
  • Step 1 - Select import file
  • Step 2 - Drag and drop, paste a URL or use select file button to choose your file. All extension files are automatically detected by Galaxy.
  • Step 3 - Click on start to begin. The importation may take some time in function of your file size.
RGV data processing
  • Step 1 - To convert your files for RGV genome browser, select RGV data processing in RGV tools tab.
  • Step 2 - Choose your file. By default, Galaxy select one of compatible file uploaded.
    RGV data processing convert bedgraph, BigWig, Wig and Bam files from specific genome release to a BigWig file from another genome release.
  • Step 3 - Choose your input genome release.
    Another release available soon.
  • Step 4 - Choose your output genome release. Select execute.
    During conversion a new tab on your right appear. The color of this tab show you the state of your job. Green : complet, Yellow : working, Red : error.
    Only genome release indexed in RGV are available.
  • Step 5 - Working done : download file by selecting download button in the process tab.
