Prepare file to RGV plot format

R script to format files and open them under RGV. This script requires the gene expression file (csv, tsv), the sample-group association file (csv,tsv), and the gene symbol-gene IDs mapping file (if gene symbol used for the expression file) (gff, csv, tsv).

usage: ./Step_0_RGV_format_file.R [-h] --rpkm RPKM_TABLE --loc LOC_TABLE
			[--map MAP_TABLE] [--gtf GFF_FILE]
			[--out OUTFILE] [-v] [-q]

	This script format file in order to visualize it in RGV gene-level and single-
	cell tool.

	optional arguments:
	-h, --help         show this help message and exit
	--rpkm RPKM_TABLE  RPKM genes table
	--loc LOC_TABLE    Cluster spacial localisation file
	--map MAP_TABLE    Gene symbol to gene ID csv file
	--gtf GFF_FILE     Ensembl gtf version associated to gene symbol
	--out OUTFILE      Output file
	-v, --verbose      Print extra output [default]
	-q, --quietly      Print little output

Requires library(argparse) and library(tools)
