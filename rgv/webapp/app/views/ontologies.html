Browse TOXsIgN ontologies

The TOXsIgN ontologies tool allows users to browse the ontologies used to describe data in TOXsIgN. This feature is especially relevant for filling the excel template used to insert data in TOXsIgN.
This tool use NCBI OntologyTree widget from BioPortal. This widget is an embeddable Flex application that allows a user to interact with a tree browser for ontologies. The widget is configured to display a list of all the ontologies in TOXsIgN, allowing the user to select which one to display, or can be setup to view a particular ontology.
The ID associated to your research term is return in the information box (bottom of the page). If you use this tool to fill the TOXsIgN excel template, please copy/paste the result in your file (NCBITaxon: XXXX).
